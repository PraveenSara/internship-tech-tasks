Agile is a way of developing software step by step instead of build a complete software at once. The steps involved or plan, design, develop, test and improve in small parts again and again. Works are done in small duration is called sprints (2-4 weeks).

GIT solves the problem of version control by tracking changes in code, enabling team collaboration, preventing data loss and managing multiple version.

API is used to enable communication between different software components, integrate 3rd party services. It act as a intermediate for two components helps to share data and get data using command like GET,PUT.
